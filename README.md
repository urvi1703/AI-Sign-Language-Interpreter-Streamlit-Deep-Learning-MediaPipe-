# 🤟 AI Sign Language Interpreter

An end-to-end deep learning-based web application that translates sign language gestures into text and speech in real time.

## 🚀 Features

* Real-time sign detection using camera
* Upload image or video
* Sentence-level interpretation
* Text-to-speech output
* English to sign language translation
* Mobile-friendly Streamlit UI
* Supports future extension to Indian Sign Language

## 🧠 Tech Stack

* Python
* Streamlit
* MediaPipe
* TensorFlow / PyTorch
* OpenCV
* NLP for sentence correction

## 📌 Project Motivation

This project aims to improve accessibility for the deaf and hard-of-hearing community by building an intuitive and scalable AI-powered sign language interpreter.
